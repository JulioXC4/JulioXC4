## About Me

I'm a Software Engineering student with hands-on experience building enterprise-grade backend systems. My foundation is in backend development, but I'm actively growing toward Full Stack — I've shipped projects spanning mobile apps, REST APIs, and web interfaces.

I recently completed an internship at **Devdatep Consulting** where I contributed to **EmiteFacil**, a real-world electronic invoicing platform integrated with Peru's SUNAT tax authority. Working on a system that businesses rely on daily gave me a strong appreciation for clean architecture, reliable integrations, and code that other people actually have to maintain.

I enjoy understanding problems end-to-end — from the business requirement down to the database query. I care about writing clean, documented code and designing systems that scale with purpose, not just systems that work today.

Currently exploring cloud infrastructure, DevOps practices, and system design patterns to round out my development perspective.

## Professional Experience

### Backend Developer Intern · Devdatep Consulting
**2025 · Lima, Peru**

Contributed to the development of **EmiteFacil**, a SaaS electronic invoicing system for Peruvian businesses.

- Designed and implemented RESTful APIs using **Laravel 11** and **PHP 8.2**, following Service-Oriented Architecture (SOA) with Repository and DTO patterns
- Integrated with **SUNAT** (Peru's national tax authority) for UBL-compliant XML invoice generation
- Built client validation features using third-party APIs for RUC and DNI verification
- Implemented authentication using **Laravel Sanctum** and contributed to product catalog, invoice series management, and PDF generation
- Participated in architecture design decisions, API documentation, and code review processes
